## 1.3. Тема проекта
Проектирование и разработка информационной системы обмена книгами между пользователями «BookSwap»

## 1.4. Цель проекта
Разработка веб-платформы, обеспечивающей поиск, обмен и учёт книг между пользователями на основе геолокации и системы рейтингов.

## 1.5. Задачи проекта
1. Провести анализ предметной области и существующих аналогов
2. Спроектировать архитектуру системы и базу данных
3. Разработать модели бизнес-процессов в нотациях IDEF0, DFD, BPMN
4. Создать UML-диаграммы структурного и поведенческого типов
5. Спроектировать ER-диаграмму и реализовать базу данных PostgreSQL
6. Разработать серверную и клиентскую части приложения
7. Провести тестирование и документирование системы

## 1.6. Аннотация
Проект представляет собой веб-платформу для обмена книгами между пользователями. Система позволяет добавлять книги в личную библиотеку, искать интересующие издания у других пользователей поблизости, отправлять запросы на обмен и оценивать участников после завершения сделки. Реализованы функции геолокации для поиска ближайших предложений, система рейтингов для повышения доверия и личный кабинет с историей обменов.
